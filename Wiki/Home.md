# ToggleSprint Documentation

Welcome to the ToggleSprint project documentation. ToggleSprint is a lightweight, high-performance utility mod for Minecraft (Fabric) designed to provide a persistent sprinting toggle without altering core game physics.

## Project Structure

This documentation covers the technical and functional aspects of the mod:

- [Installation](Installation.md): Requirements and setup instructions.
- [Compatibility](Compatibility.md): Supported versions and mod interactions.
- [Usage](Usage.md): Configuration and default keybindings.

## Core Objective

The primary goal of ToggleSprint is to improve movement efficiency for long-distance travel and combat while adhering to vanilla Minecraft mechanics. It functions strictly client-side, making it safe for use on most multiplayer networks.
