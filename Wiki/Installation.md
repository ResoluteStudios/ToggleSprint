# Installation Guide

To use ToggleSprint, ensure you have the correct mod loader and API installed for your Minecraft version.

## Core Requirements

1. **Fabric Loader**: Version 0.18.6 or higher is recommended.
2. **Fabric API**: Must be installed in the `mods` folder.
3. **Java Runtime**: Java 17 or higher for Minecraft 1.20+ and Java 21 for 1.21.x.

## Setup Instructions

1. Download the correct `.jar` version from the [Releases](https://github.com/ResoluteStudios/ToggleSprint/releases) page.
2. Locate your Minecraft installation directory (e.g., `%appdata%/.minecraft`).
3. Move the downloaded JAR into the `mods` directory.
4. Launch the game using the Fabric profile.

## Verification

The mod is functioning correctly if a new keybinding category labeled `togglesprint:togglesprint` appears in the Minecraft Controls menu.
